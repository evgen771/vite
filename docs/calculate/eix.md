## Команда eix

`eix` — это набор консольных утилит для максимально быстрого поиска и навигации по дереву пакетов (`Portage`) в операционных системах `Gentoo`, `Calculate Linux` и их производных.

Она работает значительно быстрее стандартной команды `emerge --search`, так как использует предварительно созданный бинарный кэш вместо прямого обращения к тысячам текстовых файлов `ebuild`. 

## Основные возможности

* Мгновенный поиск: Поиск пакетов по названию, описанию, категории или лицензии.
* Сравнение версий: Наглядное отображение установленных и доступных в дереве версий программ.
* Работа с оверлеями: Поддержка внешних репозиториев (overlays).
* Фильтрация: Возможность искать только установленные пакеты, пакеты с определенными USE-флагами и т.д..

## Популярные команды

   1. eix <название> — поиск пакета по имени.
   2. eix -I — показать список всех установленных пакетов.
   3. eix-update — обновление локального бинарного кэша (нужно запускать после каждого обновления дерева Portage через emerge --sync).
   4. eix-sync — комбинированная команда: синхронизирует дерево (как emerge --sync) и сразу обновляет кэш eix.
   5. eix -S <слово> — поиск по описанию пакета.

## Установка

Пакет обычно находится в категории `app-portage/eix`.

`emerge app-portage/eix`  
`emerge -av eix`

*В Calculate уже предустановлена*
