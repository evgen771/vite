# Что такое VitePress?

VitePress — это генератор статических сайтов (SSG), предназначенный для создания быстрых веб-сайтов, ориентированных на контент. 

# В двух словах.

VitePress берёт ваш исходный контент, написанный в формате Markdown , применяет к нему тему и генерирует статические HTML-страницы, которые можно легко развернуть где угодно.
