В менеджере пакетов emerge для Gentoo Linux цвета, которыми отображаются USE-флаги, означают следующее:

- Зелёный — включение при пересборке или обновлении (если пакет установлен), удаление флага (тогда он со знаком `-`, если пакет уже был с ним установлен).
- Синий — флаг неактивен (не добавляется и не удаляется).
- Красный — включение (если устанавливается новый пакет).
- Жёлтый со знаком «%» — выделены изменения в наборе флагов пакета. Флаги, которые больше недействительны (удалены) для новой версии пакета, берутся в скобки, новые (добавленные) флаги, появившиеся для новой версии, — без скобок.
- Салатовый — флаги, которые будут активированы, например, при переустановке или обновлении. 
    
## Следующие символы прикреплены к флагам USE для указания их стату

- `-`     префикс       не включен (отключен или удален)
- `*`     суффикс       переход в или из включенного состояния
- `%`     суффикс       недавно добавлен или удален
- `()`    циркумфикс    принудительно, замаскирован или удален  

- --verbose (`-v`) - Указывает emerge работать в подробном режиме  

В настоящее время этот флаг заставляет emerge выводить ошибки GNU info, если таковые имеются, и показывать флаги USE, которые будут использоваться для каждого пакета при имитации.
