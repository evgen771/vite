## Как переключить раскладку клавиатуры?

Раскладка клавиатуры переключается клавишей `Caps Lock`  
Регистр переключается через `Shift+Caps Lock`  

## Как получить права root?

Выполните в консоли `su` после чего наберите пароль `root`-а

## Как установить проприетарный видеодрайвер Nvidia/ATI?

Для установки `Nvidia` выполните:
```
emerge nvidia-drivers
cl-setup-video --video=nvidla
```
для `ATI`:
```
emerge ati-drivers
cl-setup-video --video=fglrx
```
## Как перейти на сборку пакетов из исходников?

По умолчанию, если не были изменены `USE`-флаги или маски, устанавливаются бинарные пакеты программ.

Перейти глобально на сборку пакетов из исходного кода можно раскомментировав в файле `/etc/portage/make.conf/custom строчу`:
```
FEATURES="-getbinpkg"
```
Для выборочной компиляции одного или нескольких пакетов, добавьте в тот же файл строку:
```
EMERGE_DEFAULT_OPTS="${EMERGE_DEFAULT_OPTS} --usepkg-exclude='список пакетов через пробел'"
```

Для того, чтобы видеть лог сборки, а так же исключить возможные ошибки, связанные с параллельной сборкой пакетов. раскомментируйте в этом же файле строку:

```
EMERGE_DEFAULT_OPTS="${EMERGE_DEFAULT_OPTS} --jobs=1"
```
